# 🚀 Job Portal Web Application

## 📌 Project Overview

This project is a **full-stack Job Portal Web Application** built using Python and Flask. It allows users to register as job seekers or employers, search and apply for jobs, and manage job listings efficiently.

The application demonstrates core backend development concepts such as authentication, session management, database integration, and dynamic content rendering.

---

## ✨ Features

### 👤 User Roles

* 🧑‍💻 **Job Seekers**

  * Register & Login
  * Search jobs
  * Apply for jobs

* 🏢 **Employers**

  * Post job listings
  * Manage jobs

* 🔐 **Session-based Authentication**

---

### 🔧 Core Functionalities

* 🔐 Secure Login & Registration (Session-based)
* 📄 Job Posting (Title, Description, Salary, Location, Company)
* 🔎 Job Search (Title & Location filters)
* 📥 Job Application system
* 🗄️ SQLite Database integration
* 🎨 Responsive UI using Bootstrap
* 🔁 CRUD Operations

---

## 🛠️ Tech Stack

* **Backend:** Python, Flask
* **Frontend:** HTML, CSS, Bootstrap
* **Database:** SQLite
* **Templating Engine:** Jinja2
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/job-portal.git
cd job-portal
```

### 2️⃣ Create virtual environment

```bash
python -m venv venv
venv\\Scripts\\activate
```

### 3️⃣ Install dependencies

```bash
pip install flask
```

### 4️⃣ Run the application

```bash
python app.py
```

### 5️⃣ Open in browser

```
http://127.0.0.1:5000/
```
